Client activation

How offline activation works

The activation flow stays human, controlled, and traceable. The software is installed locally, then EquiSaaS BD issues the matching license after the client shares the machine Hardware ID.

  1. Install the Windows setup on the client machine.
  2. Open the software and copy the Hardware ID shown inside the app.
  3. Send the Hardware ID to EquiSaaS BD for verification.
  4. Receive the License Key from EquiSaaS BD.
  5. Paste the License Key and activate the software offline.
